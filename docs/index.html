FAQ

SAPの専門知識がなくても使えますか？
+
はい。ERPilotはチャット形式で業務ヒアリングを行い、AIがSAP標準機能へのマッピングを自動実行します。ユーザーはSAPのT-codeや技術的な知識がなくても、業務要件を日本語で話すだけでFit/Gap分析を開始できます。
既存のSAPコンサルと併用できますか？
+
はい。ERPilotはSAPコンサルの工数を60〜70%削減する補助ツールです。現在のコンサルチームとの並走を前提に設計されており、コンサルが高付加価値業務に集中できるよう、定型作業（Fit/Gapドキュメント作成・テストケース作成・トレーニング資料作成）をAIが代替します。
2027年問題の移行スケジュールに間に合いますか？
+
ERPilotのFit/Gap分析は従来2〜3ヶ月かかる作業を2日で完了します。今すぐ始めれば、2027年末のSAP ECC保守終了に向けた移行計画を大幅に前倒しできます。まずは無料トライアルでFit/Gap分析をお試しください。
業務データは安全ですか？AIに学習されませんか？
+
業務データはAIモデルの学習には使用されません。入力された業務情報はセッション内でのみ使用され、セッション終了後は削除されます。また、すべての操作に監査証跡が付与されるため、社内セキュリティ・コンプライアンス要件にも対応しています。
対応しているSAPバージョンを教えてください。
+
SAP S/4HANA（オンプレミス・クラウド）およびSAP ECC 6.0からS/4HANAへの移行プロジェクトに対応しています。Fit/Gap分析はS/4HANAの標準機能を基準に実施します。RISE with SAPへの移行案件にも対応しております。
